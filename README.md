# 💪 App Academia#Aplicativo de academia desenvolvido em Kotlin usando Android Studio.

---

## 📱 Funcionalidades

- ✅ Criar treinos por dia
- ✅ Adicionar exercícios com peso
- ✅ Registro de séries
- ✅ Histórico de treinos
- ✅ Acompanhamento de progresso corporal

---

## 📊 Progresso

- Peso
- Peito
- Braço
- Costas
- Perna
- Coxa

---

## 🚀 Como usar

1. Abra o app
2. Crie um treino
3. Adicione exercícios
4. Acompanhe seu progresso

---

## 📦 Download

Baixe o APK na aba **Releases** do GitHub

---

## 🛠️ Tecnologias utilizadas

- Kotlin
- Jetpack Compose
- Android Studio
- SharedPreferences
